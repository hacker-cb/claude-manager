# Project language

All durable project artifacts are written in **English** — this is not a
per-author choice:

- **Issues** — titles, bodies, comments.
- **Pull requests** — titles, descriptions, review comments.
- **Documentation** — `README.md` and everything under `docs/`.
- **Code** — comments, identifiers, and commit messages.

English is the single source of truth for artifact language here, and overrides
any personal default that would pick another language for these surfaces.
